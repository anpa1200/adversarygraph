# AdversaryGraph v2.6.0 Release Notes

Release date: 2026-06-19

AdversaryGraph v2.6.0 improves day-to-day investigation flow: telemetry can be
analyzed directly, extracted TTPs can be compared as layers, and the use-case
documentation now includes animated walkthrough evidence from the published
Medium use-case article.

## Changes

- Added `POST /api/analyze/log-pcap` for log/PCAP-style analysis.
- Added IOC extraction from log/PCAP text, including IPs, domains, hashes,
  PowerShell snippets, and function-like tokens.
- Added heuristic suspicious activity findings for encoded PowerShell,
  credential dumping keywords, LOLBins, persistence, remote access, archive or
  exfiltration terms, and possible web shell indicators.
- Added AI-assisted ATT&CK extraction and actor-overlap hints for log/PCAP
  analysis.
- Added a Markdown-style log/PCAP report output for analyst review and export
  workflows.
- Added AI Analysis mode switching between CTI report analysis and log/PCAP
  analysis.
- Added actions to inject extracted analysis TTPs into Navigator and compare
  them as a separate ATT&CK layer.
- Added multi-layer Navigator comparison support with saved layers, imported
  layers, layer colors, overlap tags, and layer controls.
- Expanded Discover page actions for common workflows and platform self-test.
- Updated all use-case drafts to v2.6.0, removed draft headers, and embedded
  local animated GIF walkthroughs where the published Medium article provided
  GIF assets.
