# AdversaryGraph v2.5.8 Release Notes

Release date: 2026-06-19

AdversaryGraph v2.5.8 adds per-IOC enrichment detail pages so analysts can open
one indicator and review every stored source, enrichment value, actor link, and
ATT&CK mapping from a single workflow.

## Changes

- Added `GET /api/ioc/library/{id}/detail`.
- Added `/ioc-library/:id` in the UI.
- Added clickable IOC values in the IOC Library.
- Added clickable IOC values and an `Open detail` action in actor/group IOC tabs.
- Added source metadata, raw enrichment values, mapped TTPs, actor links, source
  report links, and raw JSON to the IOC detail view.
- Added clickable pivots into Navigator, ATT&CK Group Library, external source
  reports, and IOC Library search.
- Updated README, full guide, user guide, release notes, and release summary.

## Upgrade

```bash
git pull
docker compose up -d --build
```
