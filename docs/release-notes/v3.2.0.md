# AdversaryGraph v3.2.0 Release Notes

AdversaryGraph v3.2.0 is the Malware Analysis module documentation and release
marker update.

## Added

- Dedicated Malware Analysis module documentation with a prominent
  under-construction safety disclaimer.
- Module workflow documentation for ZIP and raw-file intake, first analysis,
  entropy and PE-header triage, string extraction, smart string analysis, AI
  string analysis, unpacking, obfuscation review, clickable IOCs, clickable TTPs,
  and runtime-debug gating.
- Release summary for the v3.2.0 MalwareGraph integration milestone.

## Updated

- README release badge, current release link, Malware Analysis capability text,
  and project maturity release scope now point to v3.2.0.
- Backend API version, frontend package version, lockfile version, root
  `VERSION`, and documentation health-check examples are updated to v3.2.0.
- Malware Analysis architecture documentation now carries the same
  under-construction safety warning as the repository landing page.

## Safety

- The module remains documented as under construction.
- Runtime debugging remains disabled unless an operator explicitly enables
  `MALWAREGRAPH_ENABLE_DYNAMIC_DEBUG=true` in a disposable, network-isolated
  profile.
- AI and automated malware-analysis output remains analyst-assistance data that
  requires manual validation.

## Validation

- Backend version module compiles with Python.
- Frontend production build completes successfully.
- Git whitespace checks pass.
