# AdversaryGraph v2.1.1 Release Notes

Release date: 2026-06-18

AdversaryGraph v2.1.1 is the rename, compatibility, and deployment validation
release for the v2.1 line. It keeps the v2.1 sector relevance and IOC
intelligence feature set, but publishes the project under the canonical
AdversaryGraph name and verifies a clean Docker deployment from a fresh clone.

## Major Changes

- Renamed the project from the previous name to **AdversaryGraph** across the
  application, repository metadata, Docker defaults, docs, screenshots,
  generated assets, publication drafts, GitHub profile links, and ecosystem
  references.
- Renamed the GitHub repository to
  `https://github.com/anpa1200/adversarygraph`.
- Renamed the documentation repository to
  `https://github.com/anpa1200/adversarygraph-docs`.
- Added legacy URL compatibility on `1200km.com` so old public links continue
  to route to the new AdversaryGraph pages, docs, article, and media assets.
- Updated ecosystem links across connected 1200km projects to point to the new
  AdversaryGraph project hub, docs, article, and repository.
- Fixed the embedded ATLAS docs nginx fallback to avoid pre-build redirect-loop
  errors during fresh Docker startup.

## Feature Set Included From v2.1

This release includes the full v2.1 feature set:

- Sector Intelligence page for actor relevance by sector, region,
  technology/environment, activity window, TTP depth, and evidence quality.
- MISP Galaxy threat-actor sync for sector, region, motivation, origin, alias,
  and evidence observations.
- IOC Intelligence model for sources, indicators, and actor links.
- ThreatFox, AlienVault OTX, custom JSON/CSV/TXT feeds, manual IOC import, and
  uploaded report IOC extraction.
- Actor IOC tabs with IOC counts, freshness, source evidence, confidence, and
  CSV export.
- Centralized IOC sync controls in Reference Sync.
- Searchable A-Z multi-select filters for sectors, regions, and
  technologies/environments.
- Sector Intelligence shortcuts to actor profile, TTP view, IOC view, and
  Navigator overlay.
- MITRE ATT&CK Enterprise, Mobile, ICS, and MITRE ATLAS sync.
- STIX 2.1 export for OpenCTI report/TTP workflows.
- Local LLM support through OpenAI-compatible endpoints.
- DFIR example metadata, report analysis workflow, and comparison workflow.
- Analysis session history with delete support.
- Review status colors for suggested, accepted, rejected, and needs-evidence
  mappings.

## Upgrade Notes

Existing clones of the old repository should move to the renamed remote:

```bash
git remote set-url origin https://github.com/anpa1200/adversarygraph.git
git pull
docker compose up -d --build
```

GitHub also serves repository redirects from the old repository URL to the new
one.

Optional IOC provider keys remain local-only:

```env
THREATFOX_AUTH_KEY=
OTX_API_KEY=
```
