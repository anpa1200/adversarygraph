# AdversaryGraph v5.6.0 Release Notes

AdversaryGraph v5.6.0 expands the Statistics module into a tag-driven
detection-engineering analysis surface.

## Added

- Tag analytics for IOC, CVE, TTP, actor/group, report, sector, and
  cross-dataset statistics.
- Risk, confidence, region, sector, type, source, telemetry source, TLP,
  attack-vector, malware-family, freeform IOC tag, and relationship-confidence
  widgets.
- A global entity tag cloud for fast cross-dataset pressure review.
- Regression checks that assert the new Statistics tag widgets are present in
  the API catalog.

## Updated

- Statistics dataset descriptions now explain which tags are available for
  analysts.
- Statistics query execution now rolls back failed widget queries before
  continuing, so sparse or missing datasets do not hide unrelated statistics.
- Release metadata now points to v5.6.0 across backend, frontend, Helm,
  README, roadmap, security policy, and version matrix.

## Validation

- Backend statistics integration tests passed.
- Frontend lint and production build passed.
- Live local Docker validation returned populated tag widgets from the current
  database, with report-only widgets empty when no stored reports are present.
